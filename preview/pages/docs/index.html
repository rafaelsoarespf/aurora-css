Aurora CSS

Everything you need to build modern interfaces with Aurora.

Quick Start

Download the Aurora CSS files from the repository and add thecss folder to your project.

css/
├─ reset.css
├─ globals.css
├─ classes.css
└─ themes.css

Then import the Aurora stylesheets:

<link rel="stylesheet" href="./css/reset.css">
<link rel="stylesheet" href="./css/globals.css">
<link rel="stylesheet" href="./css/classes.css">
<link rel="stylesheet" href="./css/themes.css">
